# Projeto de Churn de Clientes

## 📌 Descrição
Este projeto tem como objetivo analisar o **churn de clientes** (taxa de cancelamento) utilizando **Python** e **Pandas**.  
A análise busca identificar padrões de comportamento e variáveis que influenciam a saída de clientes, permitindo a criação de estratégias para retenção.

## 🚀 Tecnologias Utilizadas
- Python 3.x
- Pandas / NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## 📂 Estrutura do Projeto
- `data/` → base de dados utilizada para análise  
- `notebooks/` → notebooks com exploração e modelagem  
- `src/` → scripts auxiliares e funções  
- `results/` → gráficos e relatórios gerados  

## 📊 Resultados
Principais insights obtidos:
- Identificação das variáveis mais correlacionadas ao churn (ex.: tempo de contrato, tipo de serviço, suporte técnico).  
- Visualizações que mostram perfis de clientes com maior risco de cancelamento.  
- Sugestões de ações para retenção, como planos de fidelização e melhoria no suporte.
